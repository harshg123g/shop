# E-Shop App (React + Vite)

A fast and modern e-commerce frontend built using React and Vite. This project is optimized for quick development with hot module replacement and clean project structure.

## 🚀 Features

* Lightning-fast development with Vite
* React for building UI components
* Tailwind CSS for styling
* Routing with React Router

## 📦 Getting Started

```bash
npm install
npm run dev
```

Then open: http://localhost:5173/

## 🛠️ Build

```bash
npm run build
```
